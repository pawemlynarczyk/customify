# Miniaturki dla produktu "Portret królowej - prezent dla Niej"

Ten folder musi zawierać **2 pliki obrazów** (miniaturki do API Segmind Faceswap):

- **krolowa-prezent-1.jpg** – obrazek stylu 1 (np. portret królowej w jednym stylu)
- **krolowa-prezent-2.jpg** – obrazek stylu 2 (np. portret królowej w drugim stylu)

**Wymagania:** format 3:4 (pionowy portret), dobra rozdzielczość. Te same zasady co dla `krolowa/krolowa-styl-1.jpg` itd.

Po dodaniu plików wdróż zmiany (np. `git add . && git push`), żeby Vercel serwował je pod:
- https://customify-s56o.vercel.app/krolowa-prezent/krolowa-prezent-1.jpg
- https://customify-s56o.vercel.app/krolowa-prezent/krolowa-prezent-2.jpg
