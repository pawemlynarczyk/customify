# OpenAI Art Style - Miniaturka

Dodaj tutaj miniaturkę stylu `openai-art-style.png` w formacie:
- Format: PNG lub JPG
- Proporcje: 3:4 (pionowy portret)
- Rozmiar: zalecany 600x800px lub większy
- Lokalizacja: `public/openai/openai-art-style.png`

Miniaturka będzie wyświetlana w selektorze stylów na stronie produktu.
